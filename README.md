# Feastly — Food Delivery Web App

Feastly is a food delivery web application inspired by platforms like Zomato and Swiggy. It allows users to browse restaurants, search for dishes, add items to a cart, and place orders through a simple checkout flow.

---

## Features

* Browse restaurants
* Search for food items and cuisines
* Add and remove items from cart
* Checkout with multiple payment options (Card, UPI, Cash on Delivery)
* Login and guest access
* Responsive user interface

---

## Tech Stack

* HTML
* CSS
* JavaScript
* SessionStorage / LocalStorage

---

## Project Structure

* index.html — Homepage (restaurant listing)
* login.html — Login page
* script.js — Application logic
* styles.css — Styling

---

## How to Run

1. Download or clone the repository
2. Open index.html in a browser
